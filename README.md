# 🍂 Autumn Speech Notes

Autumn Speech Notes is a simple, elegant **web-based voice note-taking app** built using **HTML, CSS, and JavaScript**. It uses the **Web Speech API** to convert your voice into text and lets you save, copy, and manage notes directly in your browser.

---

## ✨ Features

- 🎤 Real-time **Speech to Text**
- 💾 **Save notes** with custom titles
- 📋 **Copy notes** to clipboard
- 🗑️ Delete individual notes or **clear all**
- 💽 Notes persist using **Local Storage**
- 💄 Clean, autumn-themed **responsive UI**
- ✅ Works without any backend

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- Web Speech API  
- Browser Local Storage
